# Java Fundamentals Practice Project

## Overview
This repository contains Java programs created to practice core Java fundamentals. The focus is on:

- Loops (`for`, `while`)
- Conditional statements (`if`, `else if`, `else`)
- Basic problem-solving tasks from university exercises
- Simple console-based programs like mini games

This project is intended for learning and improving Java programming skills.

## Concepts Covered
- Loops: `for`, `while`
- Conditional statements: `if`, `else if`, `else`
- Basic arithmetic and logic
- Daily university exercises automation
- Simple console-based programs like Snakes and Ladders
